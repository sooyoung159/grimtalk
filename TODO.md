# 프로젝트 TODO 및 백로그 (Backlog)

## 📌 향후 개선 사항 (나중에 작업할 일)

- [ ] **모바일 환경 STT(음성 인식) 도입 및 내 발화 내용 텍스트로 보이기**
  - **배경**: 현재 아이폰 Safari 등 모바일 브라우저의 제약으로 인해 자체 음성 인식(`Web Speech API`)이 불가능하여 부득이하게 내 말풍선에 **"방금 친구에게 다시 말을 걸었어"** 같은 고정 문구를 사용 중임.
  - **기존의 한계점**: Kanana API의 능력을 빌려 음성을 텍스트로 바꾸려고(STT) 시도했었으나, (1) 한 턴에 API가 2번씩이나 호출되어 한도를 너무 빨리 소모하는 문제와 (2) 해당 AI 모델이 텍스트 변경 과정에서 'AI 카나나'로서의 정체성을 뿜어내어 대화를 망가뜨리는 문제가 발견되었음.
  - **향후 목표**: 나중에 무료 API 한도가 여유로워지거나, STT만을 위한 저렴한/무료 전용 API(예: 외부 Whisper API 등)가 마련된다면 **다시 내 말풍선에 내가 말한 내용이 그대로 표시될 수 있도록 복구할 것.**
